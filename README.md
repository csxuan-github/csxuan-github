### 本科——竞赛

**齐鲁软件设计大赛**

在学校没有开设Android课程之前自学Android编程语言以及编程工具Android Studio，根据分治算法中“棋盘覆盖算法”设计了一款手机棋类游戏“Lchess”，与同学一起自创出适用于游戏的“黑点检测方法”。

**大学生创新创业训练计划**

将所学的SSM框架技术融入“Lchess”游戏从而使得游戏的内部逻辑更加条例清晰，提升游戏的扩展性，降低程序的耦合度。使得通信更加流畅，采用Ajax通信技术实现页面的动态刷新，从而降低了通信过程中携带字符的长度，并最终完成各种用户端之间的相互对战。

**Lchess的人机对战**

对游戏进行研究，将Lchess的人机对战分为三个层次，其中中级算法是根据游戏的特性所自己研发中级Lchess算法，高级算法则参考了棋类游戏的α-β剪枝算法实现低版本的机器学习。

**个人博客**

根据上课所学的SSM框架内容，租借服务器并完成配置，使用Spring4、jdk8、tomcat8、Ajax、JSON以及MyEclipse等编程工具，完成了个人博客编写，最终可以实现对博客页面的增删查改等操作。

**拓展**
